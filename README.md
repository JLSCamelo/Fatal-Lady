# Fatal Lady - E-commerce
Senai - Projeto semestral -
Criação e estruturação de um e-commerce de sapatos feminios

A plataforma Fatal Lady foi concebida sob os parâmetros de excelência técnica e inovação do SENAI, representando uma solução robusta de e-commerce para o mercado de calçados femininos de alto padrão.

Este sistema adota uma arquitetura modular, garantindo escalabilidade, segurança e alta performance na experiência do usuário e na gestão de dados críticos.
